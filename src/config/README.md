## Shortcuts
### Interface
- Show/hide scoreboard: O
- Show/hide the combat interface : Q
- Switch items/golds: X
- Show/hide FPS/Ping: CTRL+F

- Spectator ATH ON/OFF: Y
- Show/hide ATH: H
- Fog of war ON/OFF: F

### Camera control
- Directed camera: D
- Select blue players: 1->5
- Select red players: A->T
- Center camera on champion: Space

### Time control
- Play/Pause: P
